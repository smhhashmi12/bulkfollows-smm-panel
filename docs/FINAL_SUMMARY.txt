Test Provider Connection:
  POST /api/admin/test-provider
  ├─ Input: { api_url, api_key, api_secret? }
  ├─ Output: { success, balance, status, message }
  ├─ Timeout: 8 seconds
  └─ Purpose: Validate credentials & fetch balance

Sync Provider Services:
  POST /api/admin/sync-provider-services
  ├─ Input: { provider_id }
  ├─ Output: { success, synced_count, message }
  ├─ Timeout: 15 seconds
  └─ Purpose: Import services from provider
